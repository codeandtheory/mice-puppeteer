# mice-puppeteer

1. Clone repo
2. Open project folder in terminal
3. $ brew install node
4.  $ node -v 
5.  $ npm install
6.  $ npm init -y
7.  $ npm i puppeteer
8.  $ npm install puppeteer jest jest-puppeteer
9.  $ npm install -g jest-cli 
10.  $ node screenshot.js (this should run the test) or $ node mice_search.js
